\subsection{Detailed Description}
Keeps track of the properties of the game map. This has nothing to do with graphics and images!

Essentially, the \hyperlink{class_game_map}{Game\+Map} object keeps track of each tile and has a tileset, which in turn is another object that keeps track of tileset information (again, nothing to do with graphics or images)
